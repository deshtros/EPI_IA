Usage : 	
1) start server
2) start one IA for player 0 or 1 (should run it from the root folder EPI_IA)
    exec : python ia_cmntkikplpy RELATIF_PLAYER_FOLDER_PATH

    RELATIF_PLAYER_FOLDER_PATH : the path to folder 0 or 1
                            ex : python ia_cmntkikplpy ./server/0
                                 python ia_cmntkikplpy ./server/1

server download link : https://github.com/groznyniko/ia_fopera
